# 🔐 Vaultix

**Vaultix** is a modern desktop password manager built with **Python** and **CustomTkinter**. It provides a clean, intuitive interface for securely storing and managing your website credentials in one place.

Designed with simplicity and usability in mind, Vaultix helps users organize passwords while protecting access through a Master Key system.

---

## ✨ Features

* 🔒 Master Key protection
* 💾 Save and organize website credentials
* 👁️ Show/Hide passwords
* 📋 One-click password copy
* 🔍 Search saved passwords
* 🗑️ Delete individual passwords
* 🧹 Clear all saved passwords
* ⚙️ Settings panel
* 🔑 Reset Master Key
* ℹ️ About section
* 🎨 Clean and modern CustomTkinter interface

---

* CustomTkinter

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/deadshot1231/Vaultix.git
cd Vaultix
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python main.py
```
